# Operations Console

Enterprise-style **Ext JS Classic** demo — grouped grid, column filters, and master-detail incident panel.

## Features

- Sortable, filterable service health grid
- Rows grouped by region
- Master-detail selection panel
- Refresh and CSV export

## Run

```bash
npm install
npm run dev
```

Ext JS 6.2 is loaded from CDN (GPL). For production use, follow [Sencha licensing](https://www.sencha.com/legal/).

## Stack

Ext JS 6.2 Classic · Grid · Grouping · GridFilters · Vite (static serve)
